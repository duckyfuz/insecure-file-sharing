# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability in this project, please report it via email to **security@kenf.dev**.

## Response Process

1. We will acknowledge your report within 48 hours.
2. We will investigate the issue and determine its impact.
3. We will work on a fix and release it as soon as possible.
4. We will publicly acknowledge your contribution (unless you prefer to remain anonymous).
